# IP Camera Integration Test URLs

## Your Camera

http://192.168.0.192:8080

## Common Stream Paths to Try

- Main video stream: http://192.168.0.192:8080/video
- MJPEG stream: http://192.168.0.192:8080/video.mjpg
- HLS stream: http://192.168.0.192:8080/stream.m3u8
- Snapshot (single frame): http://192.168.0.192:8080/shot.jpg

## IP Webcam App Specific

If using IP Webcam Android app:

- MJPEG stream: http://192.168.0.192:8080/videofeed
- Audio stream: http://192.168.0.192:8080/audio.wav
- Video (browser): http://192.168.0.192:8080/browser.html
